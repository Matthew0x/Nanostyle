# Nanostyle

## Purpose

A simple nanorc file for replacing the default keybinds and graphical style in /etc/nanorc (system wide). Sub-menus are still inconsistent.

## Installation
```
> sudo rm /etc/nanorc
> sudo mv <git-path/nanorc> /etc/nanorc
```
